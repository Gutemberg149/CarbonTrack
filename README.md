# CarbonTrack - Monitoramento de Carbono 🌿

## 📱 Sobre o Projeto

CarbonTrack é uma aplicação mobile desenvolvida para monitoramento de sequestro de carbono em propriedades rurais. A solução permite que produtores rurais cadastrem suas propriedades, desenhem polígonos no mapa para delimitar áreas e calculem automaticamente a quantidade de carbono sequestrado com base na área registrada, esse calculo de carbono pode ser vendido para empresas que  buscam compensar suas emissões de gases de efeito estufa (GEE), gerando uma nova fonte de renda para o produtor rural através de **Créditos de Carbono**.


## 📝 Descrição da Solução Global Solution

O CarbonTrack foi desenvolvido como parte da Global Solution, com o objetivo de incentivar o monitoramento ambiental e a sustentabilidade no agronegócio. A solução permite:

- **Cadastro de propriedades rurais** com informações detalhadas
- **Mapeamento georreferenciado** através de desenho de polígonos no mapa
- **Cálculo automático** de área em hectares e carbono estimado
- **Monitoramento contínuo** do total de CO₂ sequestrado
- **Histórico de atividades** com as últimas propriedades cadastradas
- **Autenticação segura** de usuários com persistência de dados

## 🛠️ Tecnologias Utilizadas

### Frontend (Mobile)
- **React Native** - Framework para desenvolvimento mobile
- **Expo** - Plataforma para desenvolvimento React Native
- **React Navigation** - Navegação entre telas
- **Axios** - Requisições HTTP
- **AsyncStorage** - Persistência local de dados
- **React Native Maps** - Mapas e geolocalização
- **Expo Linear Gradient** - Gradientes visuais
- **Expo Vector Icons** - Ícones personalizados

### Backend
- **Java Spring Boot** - API RESTful
- **Render** - Hospedagem do backend
- **PostgreSQL** - Banco de dados
